Atom es un editor de código de fuente de código abierto para macOS, Linux, y Windows1​ con soporte para plug-ins escritos en Node.js y control de versiones Git integrado, desarrollado por GitHub. Atom es una aplicación de escritorio construida utilizando tecnologías web.2​ La mayor parte de los paquetes tienen licencias de software libre y está desarrollados y mantenidos por la comunidad de usuarios.3​ Atom está basado en Electron (Anteriormente conocido como Atom Shell), Un framework que permite crear aplicaciones de escritorio multiplataforma usando Chromium y Node.js.
